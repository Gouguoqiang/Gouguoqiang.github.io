取一个好变量名来尽量少些注释, 尽量只记录思路

变量名,作用域越大越具体

空间范围防歧义: first,last是[], begin,end 是[ )

bool用前缀is,can,has,should等

拆分长表达式,可引入解释变量来拆分

```java
if line.split(':')[0].strip() == "root":
    ...
```

```java
username = line.split(':')[0].strip()
if username == "root":
    ...
```
